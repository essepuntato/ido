Abstract

This ontology defines a taxonomy of all the categories for the articles described by means of the Imperial Data Ontology (IDO, https://w3id.org/ido).

IRI: https://w3id.org/ido/ArticleCategory

This class enables the specification of the types of clothes that can be associated to the various styles in the catalogue, such as "sweater", "trousers", etc. All these types must be organised as a taxonomy.
